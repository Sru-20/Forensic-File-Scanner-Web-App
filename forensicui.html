Frequently Asked Questions

What file types can the scanner analyze?

The forensic scanner supports a wide range of file types including documents (PDF, DOC, DOCX), spreadsheets (XLS, XLSX), images (JPEG, PNG, GIF), archives (ZIP, RAR), executables (EXE, DLL), and scripts (JS, VBS, PS1).

How accurate is the threat detection?

Our multi-layered detection system combines signature-based scanning, heuristic analysis, and behavioral assessment to achieve over 98% detection accuracy with minimal false positives.

Can I scan files from cloud storage?

Currently, files need to be downloaded to your local system before scanning. We're working on direct cloud storage integration for future releases.

How long does a scan typically take?

Scan duration depends on file size and scan mode. Quick scans take seconds for most files, while deep scans of large files (100MB+) may take several minutes.
